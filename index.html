How AI SecSum works

A Trivy scan becomes a prioritized, AI-written security assessment through ten automated stages — from container scanning to executive remediation plans.

Processing pipeline

01
Trivy Vulnerability Scan Docker image is scanned and checked against CVE databases
02
JSON Report Generation Trivy produces a detailed, machine-readable JSON summary
03
Python JSON Parsing Report data is ingested and mapped to internal objects
04
Vulnerability Prioritization Sorting vulnerabilities by severity and package density
05
Prompt Formatting Structuring the metrics and vulnerabilities into an LLM prompt
06
Amazon Bedrock Reasoning Invoking the Nova Lite model to perform security analysis
